# Arkanoid

### About the project
This project is my take on the classic game [Arkanoid](https://en.wikipedia.org/wiki/Arkanoid). 
### Languages
- Java.
### How to run it 
1. Download the zip file.
2. Unzip.
3. Open "Arkanoid-main" directory in the terminal.
4. Type the following command:
    - `ant compile`
5. Then type:
   - `ant run`
